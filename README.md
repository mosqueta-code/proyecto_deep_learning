# Proyecto: Clasificador Inteligente de Imágenes de Ropa 
## (Deep Learning)

El objetivo de este proyecto es diseñar, entrenar e implementar un modelo basado en redes neuronales que sea capaz de clasificar imágenes de prendas de vestir (camisetas, zapatillas, pantalones, etc.) en sus respectivas categorías. 
Para el desarrollo de este proyecto se utiliza el dataset Fashion-MNIST.

El proyecto se desarrolla en 3 etapas evolutivas:

1. **Clasificación Binaria:** validación inicial utilizando solo dos clases.
2. **Red Densa (MLP):** implementación de una red densa aplicando técnicas de regularización y optimización.
3. **Red Convolucional (CNN):** rediseño de la arquitectura usando bloques de convolución y pooling.
